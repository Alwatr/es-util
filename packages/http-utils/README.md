# http-utils

Some constants that are categorized based on `HTTP` terminology..

## Installation

```bash
yarn add @alwatr/http-utils
```

## Usage

```typescript
import {HttpStatusCodes} from '@alwatr/http-utils';

console.log(HttpStatusCodes.Success_200_OK); // 200
```
