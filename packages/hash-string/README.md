# hash-string

A tiny TypeScript library to hash a string or number.

## Installation

```bash
npm install @alwatr/hash-string
```

## Usage

```ts
import {hashString} from '@alwatr/hash-string';

console.log(hashString('string'));
console.log(hashString(123));
```
