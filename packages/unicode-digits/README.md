# Unicode Digits

## Installation

```bash
yarn add @alwatr/unicode-digits
```

## Usage

```ts
import {UnicodeDigits} from '@alwatr/unicode-digits';

const unicodeDigits = new UnicodeDigits('en');
unicodeDigits.translate('۱۲۳'); // 123
```
